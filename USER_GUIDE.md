## ğŸ¨ Tips & Tricks

### For Best Results

1. **Document Quality**
   - Use text-based PDFs (not scanned images)
   - Clear, well-formatted content works best
   - Longer documents = more comprehensive output

2. **Effective Questions (Chat)**
   - Be specific: "Explain X" vs "Tell me about this"
   - Ask follow-ups for deeper understanding
   - Reference specific sections if needed

3. **Flashcard Strategy**
   - Generate multiple sets for different topics
   - Name sets clearly (e.g., "Biology Ch. 5")
   - Review regularly, not just before exams

4. **Mindmap Usage**
   - Use fullscreen for complex topics
   - Download and print for offline study
   - Explore all branches thoroughly

5. **Summary Best Practices**
   - Read summary first for overview
   - Use as study guide outline
   - Download for offline reference
